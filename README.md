## Installation

1. Clone this repository
   ```bash
   git clone git@github.com:hendraaagil/vlr-api.git
   ```
2. Install dependencies
   ```bash
   yarn # or npm install
   ```
3. Run server
   ```bash
   yarn dev # or yarn start
   ```
