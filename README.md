## Installation

1. Clone this repository
   ```bash
   git clone git@github.com:hendraaagil/vlr-api.git
   ```
2. Make `.env` file and fill the `SITE_URL`
   ```bash
   cp .env.example .env
   ```
3. Install dependencies
   ```bash
   pnpm install
   ```
4. Run server
   ```bash
   pnpm dev # or pnpm start
   ```
